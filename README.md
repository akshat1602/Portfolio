# Akshat Trivedi Portfolio

A fully responsive personal portfolio website built using HTML, CSS, and JavaScript to showcase my profile, skills, services, projects, and contact details. The website is designed with a clean dark theme, smooth navigation, interactive hover effects, and a mobile-friendly layout.

## Tech Stack

- HTML5
- CSS3
- JavaScript

## Sections Included

- Home
- About Me
- Services
- Portfolio / Projects
- Contact

## Project Structure

```bash
Portfolio/
│── index.html
│── style.css
│── images/
```
