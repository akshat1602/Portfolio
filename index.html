Stayora

A full-stack Airbnb-inspired rental listing platform built to showcase practical product development across authentication, CRUD, database relationships, maps, image hosting, search, filters, wishlist handling, booking validation, and review flows.

Stack Node.js, Express.js, MongoDB Atlas, EJS, Passport.js, Mapbox, Cloudinary, Bootstrap
Core Features Discovery, filters, wishlist, bookings, reviews, listing management

Stayora was built as a portfolio-ready full-stack project that solves the problem of discovering and managing rental stays through a more complete web flow. It demonstrates server-side rendering, user auth, database-driven listings, geocoding, image uploads, booking logic, and polished listing UX in one integrated product.

MeetSphere

MeetSphere

A real-time video meeting web application built to handle meeting creation, joining, authentication, live communication, meeting history, and responsive collaboration in a cleaner product interface.

Stack React, React Router, MUI, Node.js, Express, Socket.io, MongoDB Atlas, WebRTC
Core Features Auth, meetings, live video/audio, chat, history, responsive UI

MeetSphere demonstrates real-time application thinking through WebRTC and Socket.io-based communication layered with login flows, meeting management, history tracking, and frontend-backend separation. It shows how a production-style collaboration app can be structured across React on the frontend and Node/Express on the backend.
